Disease Diagnosis System (Prolog)

This project is a simple expert system written in Prolog.
It identifies possible diseases based on two symptoms entered by the user.
The knowledge base contains symptoms for flu, cold, covid, migraine, allergy, typhoid, dengue, and food poisoning.

How to Run

1. Open SWISH or any Prolog environment.


2. Load the file diagnose.pl.


3. Run the query:

diagnose

4. Enter the two symptoms when asked.
